**DESCRIPTION**

This project explores the relationships between daily screen time, sleep quality, and stress levels, along with other factors that can impact a person's wellbeing. The exploration is done with visualizations created with D3, along with supplementary information, that has been added to a GitHub.io website. The visualizations highlight how an increased amount of screen time is associated with higher stress levels and poor sleep quality, and how these three main factors interact. Additional visualizations highlight the comparisons between the three main groups with other factors such as occupation. The goal of this project is to display a clear, data-driven narrative about how one's screen time habits can positively or negatively impact their wellbeing, which can lead to them making better choices.

**LESSONS LEARNED**

*D3*

Doing this project helped me learn how to properly use D3. Using D3 requires the data to be structured in a specific way, especially when grouping or aggregating the data. I also learned how to choose appropriate visualization types that properly convey the data I want to share, while appealing to the audience. 

*Visualizations*

Interactive features may be more complicated to implement, but can boost user engagement and interactions while making the content easier to understand.  

*Dataset*

Many factors that I thought would significantly impact sleep, screen time, OR stress levels were not relevant and only had minimal impacts (age, gender, physical activity, even caffeine intake!). Screen time was the strongest factor that affected both stress and sleep.
